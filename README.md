## 🚀 Getting Started

Because this application accesses your webcam, **modern web browsers require it to be run on a local web server.** You cannot simply double-click the `index.html` file.

### Prerequisites
You will need a way to serve local HTML files. We recommend the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for VS Code, or Python's built-in HTTP server.
